## Python

### mutable 데이터와 immutable 데이터
- mutable 데이터는 변할 수 있는 데이터
    - list형, dictionary형
- immutable 데이터는 변하지 않는 데이터
    - int형, float형, str형, tuple형

### tuple(튜플)형 데이터
- 튜플은 element(원소)의 값을 수정할 수 없음
- 순서와 값이 중요한 곳에 사용
  
### 튜플형 데이터의 특징
- "()" 소괄호를 사용
- 리스트형 데이터처럼 자료를 만들 수 있음
- 인덱싱과 슬라이싱이 가능
- '+, *'를 사용하여 데이터를 연결하고 반복시킬 수 있음
